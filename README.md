# Sivabharathi M - Professional Portfolio

A modern, responsive portfolio website showcasing my expertise in Artificial Intelligence, Machine Learning, and Software Development.

## 🚀 Features

- **Responsive Design**: Optimized for all devices and screen sizes
- **Dark/Light Theme Toggle**: Professional theme switching with smooth transitions
- **AI-Enhanced Interactions**: Smart greetings and interactive elements
- **Animated Particles Background**: Dynamic visual effects
- **Professional Contact Form**: Direct email integration
- **Resume Download**: Downloadable professional resume
- **Smooth Animations**: Intersection Observer-based animations
- **Performance Optimized**: Lazy loading and optimized assets

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: CSS Grid, Flexbox, CSS Variables
- **Icons**: Font Awesome
- **Fonts**: Google Fonts (Inter)
- **Animations**: CSS Animations, Intersection Observer API
- **Performance**: Service Worker ready, optimized loading

## 📱 Sections

1. **Hero Section**: Dynamic typing animation with professional introduction
2. **About**: Professional summary and career objectives
3. **Skills**: Technical skills with animated progress bars and core competencies
4. **Projects**: Featured projects with detailed descriptions
5. **Education**: Academic background and certifications
6. **Contact**: Professional contact form and information

## 🎨 Design Features

- **Modern UI/UX**: Clean, professional design with attention to detail
- **Color Scheme**: Professional blue gradient with excellent contrast
- **Typography**: Inter font family for optimal readability
- **Micro-interactions**: Hover effects and smooth transitions
- **Accessibility**: Semantic HTML and ARIA labels

## 🚀 Getting Started

1. Clone or download the repository
2. Open `index.html` in a web browser
3. Or serve using a local server:
   ```bash
   python3 -m http.server 8000
   ```
4. Navigate to `http://localhost:8000`
