# Sivabharathi M - Professional Portfolio

A modern, responsive portfolio website showcasing AI/ML projects, skills, and achievements.

## Features

✨ **Modern Design**
- Clean and professional UI/UX
- Smooth animations and transitions
- Fully responsive across all devices

🎨 **Dark/Light Theme Toggle**
- Persistent theme preference
- Smooth theme transitions
- Eye-friendly color schemes

📄 **Resume Download**
- Printable HTML resume
- Easy PDF conversion (Ctrl/Cmd + P)
- Professional formatting

📧 **Contact Form**
- Direct email integration
- Mailto functionality
- User-friendly interface

🚀 **Performance Optimized**
- Lazy loading
- Smooth scrolling
- Intersection Observer animations
- Particle effects background

## Sections

1. **Hero** - Introduction with typing animation
2. **About** - Professional summary and career objectives
3. **Skills** - Technical skills and core competencies
4. **Projects** - Featured AI/ML projects
5. **Education** - Academic background and certifications
6. **Contact** - Contact information and message form

## Technologies Used

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts (Inter)

## How to Use

1. **View Portfolio**: Open `index.html` in a web browser
2. **Download Resume**: Click the download button in the Contact section
3. **Toggle Theme**: Click the moon/sun icon in the navigation
4. **Send Message**: Fill out the contact form to send an email

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Setup

Simply open `index.html` in your web browser. No build process required!

## Customization

- Update personal information in `index.html`
- Modify colors in CSS variables in `main.css`
- Adjust animations and functionality in `script.js`
- Update resume content in `resume.html`
