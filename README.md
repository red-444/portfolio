# Sivabharathi M - Professional Portfolio

A modern, responsive portfolio website built with HTML, CSS, and JavaScript featuring AI-powered animations, dark/light theme toggle, and professional design.

## Features

### 🎨 Design & UI/UX
- **Modern Professional Design** - Clean, minimalist layout with professional color scheme
- **Dark/Light Theme Toggle** - Smooth theme switching with persistent storage
- **Responsive Design** - Fully responsive across all devices (mobile, tablet, desktop)
- **Smooth Animations** - AI-powered floating elements and scroll animations
- **Interactive Elements** - Hover effects, smooth scrolling, and dynamic content

### 🤖 AI-Powered Features
- **Floating Elements** - Animated AI/ML skill tags around profile
- **Particle Effects** - Dynamic background particles
- **Typing Animation** - AI-powered typing effect for hero title
- **Scroll Animations** - Elements animate into view as you scroll
- **Performance Optimization** - Adaptive animations based on device capabilities

### 📄 Resume & Contact
- **Resume Download** - One-click resume download as PDF
- **Contact Form** - Professional contact form with validation
- **Email Integration** - Direct email functionality
- **Social Links** - LinkedIn, email, and phone integration

### 🛠️ Technical Features
- **Modern CSS** - CSS Grid, Flexbox, and CSS Variables
- **Vanilla JavaScript** - No external dependencies, pure JS
- **Local Storage** - Theme preference persistence
- **Form Validation** - Client-side form validation
- **Accessibility** - ARIA labels and keyboard navigation

## Getting Started

1. **Open the Portfolio**
   - Simply open `index.html` in any modern web browser
   - No server setup required - works locally

2. **Theme Toggle**
   - Click the moon/sun icon in the navigation bar
   - Theme preference is automatically saved

3. **Download Resume**
   - Click "Download Resume" button in the hero section
   - Opens a print dialog to save as PDF

4. **Contact Form**
   - Fill out the contact form in the Contact section
   - Validates input and opens email client

## Customization

### Personal Information
Edit the following in `index.html`:
- Name, title, and description in hero section
- Contact information in contact section
- Education details in about section
- Skills and projects in respective sections

### Styling
Modify `styles.css`:
- Color scheme in `:root` variables
- Typography and spacing
- Animation timings and effects

### Functionality
Update `script.js`:
- Contact form email handling
- Resume content generation
- Animation parameters
