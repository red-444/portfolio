## 🎨 Customization Guide

### Change Colors
Edit `main.css` lines 2-18:
```css
:root {
    --primary-color: #3b82f6;    /* Main blue */
    --secondary-color: #06b6d4;  /* Cyan */
    --accent-color: #8b5cf6;     /* Purple */
}
```

### Update Content
- Personal info: Search and replace in `index.html`
- Projects: Edit project cards in `index.html`
- Skills: Modify skill items and percentages
- Profile image: Replace `img.jpg`

### Add Sections
- Copy existing section structure
- Add navigation link
- Style in `main.css`
- Add animations in `script.js`
